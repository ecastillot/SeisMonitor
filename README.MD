# SeisMonitor

Set of routines to monitor seismic activity.

*In process.*

# Installation

## Pip
```bash
pip install SeisMonitor==0.0.4
pip install git+https://github.com/wayneweiqiang/GaMMA.git
```
## Anaconda
```bash
conda create --name seismonitor python=3.7.9
conda activate seismonitor
conda install pip
pip install SeisMonitor==0.0.4
pip install git+https://github.com/wayneweiqiang/GaMMA.git
```
